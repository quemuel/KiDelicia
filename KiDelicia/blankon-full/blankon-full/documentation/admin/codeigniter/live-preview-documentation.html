Base HTML structure

This theme is a fluid layout with two columns and or three columns. All of the information within the page content area is nested within a div with id of "page-content". The sidebar left's (column #2) content is within a div with an id of "sidebar-left". The sidebar right's (column #3) content is within a div with an id of "sidebar-right". Codeigniter installation guide

This manual explains how to installation Blankon Codeigniter on your project using Codeigniter Framework.

Step 1

Download Codeigniter

Official site : http://www.codeigniter.com/download

Step 2

Open application folder and then copy merge config, controllers, core, libraries, third_party and views folder from Blankon-Codeigniter to your Codeigniter folder.

Step 3

Copy merge Blankon assets folder to your Codeigniter project

Step 4

Go to config folder, open config.php and change this line

$config['base_url'] = 'Your url website';
Step 5

Open browser and testing Codeigniter system.

Example URL browser : http://localhost/blankon-fullpack-admin-theme/1.0.7/production/admin/codeigniter

Congratulations! you can now starting Blankon Codeigniter app :D
